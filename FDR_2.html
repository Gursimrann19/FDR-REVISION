5.How to use flex ? apply display: flex to the parent body. use jutify-content: center to put items on the helf of main exis and align items to put items on the half of cross axis.
Use flex-direction and flex-wrap. use flex flow to set both the things in one go.
align items vs align content. use align items for one line. use align content for centering of multiple lines
Use gap: 30px; on parent body. to define the gap between flex items only.
all the above mentioned properties are defined on the parent body
all the below mentioned prp are defined on flex items.Use .order-1 { order: 1 }; the default order is 0. higher number comes later.
use flex-grow and flex-shrink.(like col-span and row-span in table)
flex-shrink is used when u decrease the width of screen
use align-self to overwrite the configuration of one flex item.
6.How to use grid ? grid is used for converting an container into a grid consisting of rows and columns.
on the parent, write display:grid grid-template-column: 120px 100px; this means u will form 2 columns of 120px and 100px . you can also say [nameline1] 120px [nameline2] 100px the height of rows will depend on child's height
what is grid track? space between 2 parallel lines. what is grid area ? space between 4 adjacent lines
item property : you can define grid-row: start/end for any particular component. eg. .cclass4 {grid-row: 1/3; }
you can also use justify self on grid items. and justify-items on grid parent. These properties are applied within a grid cell.
Note

While defining grid-columns on parents grid. instead of giving 120px 100px you can also give 'fr' .
grid- column: 1fr 2fr 1fr means divide the parent container into 3 fractions with center twice in width.

Note
you can also use grid-template-area on the parent. use: 
        
            .conatainerg { 
                grid- template- areas:
                "nav nav nav"
                "sidebar content content"
                "footer footer footer";

                grid-template- columns : 1fr 2fr 1fr
                or 
                grid-template-columns : repeat(4,1fr)
                OR 
                grid-template -column : repeat(4, minmax(100px,1fr));

            }
            .item-11 {
                grid- area : nav ;
            }
            .item-22{
                grid- area: sidebar;
            
            }

            .item-55{
                grid- area: footer;
            }

Use gap in flex and grid instead of padding.

container --> grid --> item --> text

use align-content , justify-content, when u want to center the grid in the container. YOU CAN ALSO USE PLACE-CONTENT INSTEAD. IT ENCAPSULATES BOTH.

use align-items, justify-items, when u want to center the items within grid

juse align-self , justify-self , when u want to center the text within the item. this is applied on the item body. not on the parent body

Transition and Transformation

7. there are 4 transition features . property, duration, delay and timing function. how to use it 
        


    ---------------------------HTML---------------------------------------------
    < div class="ccontainer">Hello iam container
        < div class="ccard" id="itransition"> Hello iam a card < /div>
    < /div>  
    
    ----------------------------CSS--------------------------------------------
.ccard{
    background-color: red;
    height: 300px;
    width: 220px;
    transition-property: color background-color transform ;
    transition-duration: 5s ;
    transition-timing-function: ease-in-out ;
    transition-delay: 1s ;
}

#itransition{   
    background-color: black;
    transform: translateX(10vw) translateY(2vh);
    color: white;
    width: 300px;
    height: 110px;
}
8.Similarly you can use animation.you dont need to create a separate class as needed for transitions. and associate that animation with the component desired. in animation you write @keyframes { to {....} from{...}}
        
 ---------------------------HTML-------------------------------------------

    }
}

OBJECT FTI AND OBJECT POSITION

9.Use object fit : cover or contain on an image to fit it in a container. cover means crop. contain means extra space. the default image fit is fill which streches the image to fit it in a cover.
10. use object position to retain any one corner of the image if its a very important part of the image.
Use background image to set an image as an bg. by default the background image repeats if it doesnt fit into the container or the screen.
for images contrast,opacity is always [0,1] , invert is 0 or 1. use filter: blur/contrast, brightness
